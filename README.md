# REGALOPE-Temporal-
Proyecto de Emprendimiento Temporal
RegaloPe, una plataforma web diseñada para conectar emocionalmente a la diáspora peruana con sus familiares en el país. Identificando que el envío de regalos físicos desde el extranjero es un proceso costoso, lento y logísticamente complejo, RegaloPe ofrece una solución digital basada en tarjetas virtuales altamente personalizables
